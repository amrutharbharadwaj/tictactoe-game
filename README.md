# Tic-Tac-Toe: 2-Player and Unbeatable AI

A modern, responsive Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against a friend on the same device or challenge the Unbeatable AI (Minimax Algorithm) to test your skills.

## Features

- 2-Player Mode: Classic local multiplayer. Take turns on the same device.
- Unbeatable AI Mode: Play against an AI powered by the Minimax algorithm. It never loses.
- Fully Responsive: Works perfectly on desktop, tablet, and mobile devices.
- Modern UI: Clean design with smooth animations and a win/draw detection system.
- No Dependencies: Pure JavaScript, no frameworks or libraries required.

## Tech Stack

- HTML5: Semantic structure
- CSS3: Styling, Flexbox/Grid, Animations
- JavaScript (ES6+): Game logic, AI algorithm, DOM manipulation

## How the AI Works

The "Unbeatable" mode uses the Minimax Algorithm. This recursive algorithm simulates all possible future moves to determine the optimal step, ensuring the AI either wins or forces a draw.
